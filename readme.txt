== Description ==

The soup.io feed widget plugin allows you to place a feed from your soup.io account in the sidebar. Simply set your soup.io username in widget settings and place the widget in your side bar.

== Installation ==

1. Download the zipped folder
2. Upzip it
3. Copy the soup.php file into your plugins folder
4. Drag the widget to your side bar.
5. Set your soup.io username in widget settings.

== Frequently Asked Questions ==

= Why use Soup.io Feed =

This is the first widget to feed your Soup.io blog to your Wordpress blog.
